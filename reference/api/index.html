Complete API

src.datafolio

datafolio: Lightweight wrapping of dataframes, models, and metadata to track analyses.

Modules:

  • accessors –

    Data accessors for convenient item access.

  • base –

    Base classes and utilities for datafolio handler system.

  • cache –

    Local caching system for cloud-based DataFolios.

  • cli –

    Command-line interface for DataFolio.

  • display –

    Display formatting for DataFolio describe() output.

  • folio –

    Main DataFolio class for bundling analysis artifacts.

  • handlers –

    Data type handlers for DataFolio.

  • metadata –

    Metadata dictionary with auto-save functionality.

  • readers –

    Readers for different table formats.

  • storage –

    Storage backend for file I/O operations.

  • utils –

    Utility functions for datafolio.
